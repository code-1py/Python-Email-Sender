# Python-Email-Sender
A simple Python script to **send emails automatically** using Gmail's SMTP server.  
This project demonstrates how to send emails programmatically with Python.

---

## Features

- Send emails to **single or multiple recipients**
- Customize **subject** and **body** of the email
- Securely login using **Gmail app password**
- Handles errors and displays informative messages

---

## Prerequisites

- Python 3.x installed
- Gmail account with **app password** enabled
- Libraries used:
  ```bash
  pip install secure-smtplib email

# Note: smtplib and email are included in Python standard library
